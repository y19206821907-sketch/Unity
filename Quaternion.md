# 四元数的逆为什么是共轭除以模长？

## 1. 逆的定义
不管是数字、矩阵还是四元数，逆的定义都只有一个：原数 × 逆 = 1（单位元）。

$$q \times q^{-1} = 1$$

## 2. 四元数与共轭
- 四元数定义：
$$q = (w, x, y, z)$$
- 共轭的定义是把虚部取反：
$$\overline{q} = (w, -x, -y, -z)$$

## 3. 共轭的神奇作用
当你把一个四元数乘以它的共轭，结果会变成一个纯实数：
$$q \times \overline{q} = w^2 + x^2 + y^2 + z^2$$

这个数就是四元数的模长平方，记作：
$$\|q\|^2 = w^2 + x^2 + y^2 + z^2$$
